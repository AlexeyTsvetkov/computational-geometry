Build
=========
Run in terminal:
```bash
cmake --target all .
make
```

Usage
==========
Run in terminal:
```bash
./program
```

Expected input format
=========
```bash
3      // number of polygon vertices  
(0, 0) // polygon vertex: (x, y)     
(3, 0)
(0, 2)
3       // number of test points 
(0, 1)  // test point: (x, y)    
(1, 1)
(2, 1)
```

Expected output format
========
```
yes
yes
no
```
